Building Modular Campaigns

Created an overview feature that summarized modular steps that needed to be completed before running a campaign. Users had the need to review required elements at a glance without having to revisit each object individually. This was centered around the concept of a dynamic, readable sentence that explained the connection between each component and its status. This design validated each step and provided a flexible completion order that a traditional linear wizard might not.
